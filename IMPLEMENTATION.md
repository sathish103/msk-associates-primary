## 🆘 Troubleshooting

### **Issue: Images not showing**
- [ ] Verify images in `assets/images/` folder
- [ ] Check filename spelling (case-sensitive)
- [ ] Refresh browser (Ctrl+F5)
- [ ] Clear cache (Ctrl+Shift+Delete)

### **Issue: Styles look broken**
- [ ] Verify `assets/css/style.css` exists
- [ ] Check file permissions
- [ ] Hard refresh (Ctrl+Shift+R)
- [ ] Try different browser

### **Issue: JavaScript not working**
- [ ] Verify `assets/js/main.js` exists
- [ ] Check browser console (F12)
- [ ] Hard refresh (Ctrl+Shift+R)
- [ ] Try different browser

### **Issue: Mobile menu not responsive**
- [ ] Resize browser window
- [ ] Or use mobile device
- [ ] Check DevTools device emulation
- [ ] Verify viewport meta tag in HTML

### **Issue: Chat/WhatsApp buttons not working**
- [ ] Update phone numbers in code
- [ ] Check URL format in href attributes
- [ ] Ensure WhatsApp is installed on device
- [ ] Check email client configured
